# Machine Learning for Kidney Disease Detection

## Overview
This project applies machine learning algorithms to detect Chronic Kidney Disease (CKD)
using the UCI Machine Learning Repository dataset.

## Author
- Gnan Deep Koneru

## Dataset
UCI Chronic Kidney Disease dataset — includes clinical and laboratory features
such as age, blood pressure, blood glucose, serum creatinine, and more.

## Algorithms Used
- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Naive Bayes
- Bagging
- Boosting ← Best performer (100% accuracy)
- XGBoost
- Neural Network (TensorFlow/Keras)

## Results
| Classifier         | Accuracy | Precision | Recall | F1 Score |
|--------------------|----------|-----------|--------|----------|
| Logistic Regression| 0.9875   | 0.981     | 1.0    | 0.990    |
| Random Forest      | 0.975    | 0.963     | 1.0    | 0.981    |
| Boosting           | **1.0**  | **1.0**   | **1.0**| **1.0**  |
| XGBoost            | 0.95     | 0.962     | 0.962  | 0.962    |
